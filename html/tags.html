Glossário de HTML5

1. <div>

Descrição: Um contêiner genérico para agrupar elementos.

<div class="container">
    <h1>Bem-vindo ao meu site!</h1>
</div>

2. <p>

Descrição: Define um parágrafo.

<p>Este é um parágrafo de texto.</p>

3. <a>

Descrição: Define um hyperlink.

<a href="https://www.exemplo.com">Visite nosso site!</a>

4. <img>

Descrição: Insere uma imagem.

<img src="imagem.jpg" alt="Descrição da imagem">

5. <h1> a <h6>

Descrição: Define cabeçalhos, de <h1> (mais importante) a <h6> (menos importante).

<h1>Título Principal</h1>
<h2>Subtítulo</h2>

6. <ul> e <ol>

Descrição: Define listas não ordenadas (<ul>) e ordenadas (<ol>).

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

<ol>
    <li>Primeiro item</li>
    <li>Segundo item</li>
</ol>

7. <li>

Descrição: Define um item de lista.

<li>Item de lista</li>

8. <table>

Descrição: Define uma tabela.

<table>
    <tr>
        <th>Nome</th>
        <th>Idade</th>
    </tr>
    <tr>
        <td>João</td>
        <td>30</td>
    </tr>
</table>

9. <form>

Descrição: Define um formulário para entrada de dados.

<form action="/submit" method="post">
    <label for="nome">Nome:</label>
    <input type="text" id="nome" name="nome">
    <input type="submit" value="Enviar">
</form>

10. <input>

Descrição: Define um campo de entrada.

<input type="text" placeholder="Digite seu nome">

11. <button>

Descrição: Define um botão clicável.

<button type="button">Clique aqui</button>

12. <span>

Descrição: Um contêiner genérico para texto em linha.

<span class="destaque">Texto destacado</span>

13. <section>

Descrição: Define uma seção em um documento.

<section>
    <h2>Seção 1</h2>
    <p>Conteúdo da seção 1.</p>
</section>

14. <article>

Descrição: Define um conteúdo independente que pode ser distribuído.

<article>
    <h2>Título do Artigo</h2>
    <p>Conteúdo do artigo.</p>
</article>

15. <footer>

Descrição: Define o rodapé de um documento ou seção.

<footer>
    <p>© 2023 Meu Site</p>
</footer>

16. <header>

Descrição: Define o cabeçalho de um documento ou seção.

<header>
    <h1>Meu Site</h1>
</header>

17. <nav>

Descrição: Define uma seção de navegação.

<nav>
    <ul>
        <li><a href="#home">Home</a></li>
        <li><a href="#sobre">Sobre</a></li>
    </ul>
</nav>

18. <aside>

Descrição: Define conteúdo relacionado, mas não essencial.

<aside>
    <h2>Links Relacionados</h2>
    <p>Conteúdo adicional.</p>
</aside>

19. <main>

Descrição: Define o conteúdo principal do documento.

<main>
    <h2>Conteúdo Principal</h2>
    <p>Este é o conteúdo principal da página.</p>
</main>

20. <canvas>

Descrição: Define uma área de desenho.

<canvas id="meuCanvas" width="200" height="100" style="border:1px solid #000000;"></canvas>

21. <video>

Descrição: Define um vídeo.

<video width="320" height="240" controls>
    <source src="video.mp4" type="video/mp4">
    Seu navegador não suporta o elemento de vídeo.
</video>

22. <audio>

Descrição: Define um conteúdo de áudio.

<audio controls>
    <source src="audio.mp3" type="audio/mpeg">
    Seu navegador não suporta o elemento de áudio.
</audio>

23. <iframe>

Descrição: Define um quadro embutido.

<iframe src="https://www.exemplo.com" width="300" height="200"></iframe>

24. <blockquote>

Descrição: Define uma citação em bloco.

<blockquote>
    <p>Esta é uma citação famosa.</p>
</blockquote>

25. <cite>

Descrição: Define o título de uma obra.

<cite>O Senhor dos Anéis</cite>
